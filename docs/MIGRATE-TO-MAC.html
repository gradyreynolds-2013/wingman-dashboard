Mac-Specific Tips

Keep Your Mac Awake

Clawdbot needs to stay running. In System Preferences: - Energy Saver ‚Üí Prevent Mac from sleeping when display is off (desktop) - Or use: caffeinate -s to keep system awake

Firewall

macOS Firewall is usually fine. Clawdbot only listens on localhost by default (no external access).

Updates

Clawdbot updates via npm:

npm update -g clawdbot
